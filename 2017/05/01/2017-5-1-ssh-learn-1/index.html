SSH框架学习之旅（一）：框架搭建

Posted on 2017-05-01 | In programming

1.导入相关jar包

（1）下载并导入spring相关jar包

进入http://repo.spring.io/release/org/springframework/spring/下载相应版本的zip文件，解压后得到jar包。复制jar包到WEB-INF/lib并build path对应的jar包。

（2）下载并导入Struts相关jar包

进入http://struts.apache.org/download.cgi#struts25101下载相应版本的zip文件，解压后得到jar包。复制jar包到WEB-INF/lib并build path对应的jar包。

（3）下载并导入hibernate相关jar包

进入http://hibernate.org/orm/downloads/下载相应版本的zip文件，解压后得到jar包。复制jar包到WEB-INF/lib并build path对应的jar包。
